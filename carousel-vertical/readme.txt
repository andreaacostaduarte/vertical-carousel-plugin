== Description ==

<p>Simple carousel with a vertical layout. Initial jQuery carousel taken from Jan Sorgalla (http://sorgalla.com)</p>

<p>Large image dimensions: 545x343px</p>
<p>Thumbnail image dimensions: 154x127</p>

== Installation ==

1. Upload `vertical-carousel` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the widget tool and drag the Vertical carousel widget to where you would like it in your template
